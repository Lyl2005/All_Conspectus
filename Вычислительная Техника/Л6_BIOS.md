# BIOS - Basis input-output system 

## Программа предназначена для первоначального запуска компьютера, настройки оборудования и обеспечения функций ввода-вывода.  
BIOS Записывается в микросхему флеш-памяти, расположенную на мат плате.  
В современных компьютерах BIOS выполняет несколько доп функций: 
1. Запуск компьютера и процедура самотестирования (post)  
Программа, расположенная в микросхеме BIOS загружается первой после включения питания. Она определяет и проверяет установленное оборудование, настраивает устройства и готовит их к работе. Если в процессе самотестирования будет обнаружена неисправность, то процедура post будет остановлена с выводом соответствующего сообщения или сигнала.  Если во время проверки все прошло успешно, самотестирование завершается и вызывается встроенная подпрограмма для загрузки ОС. 
2. Настройка параметров системы с помощью BIOS Setup 
Во время процедуры post оборудование настраивается в соответствии с параметрами BIOS, хранящиеся в специальной Cmos памяти.  
3. Поддержка функций ввода-вывода с помощью программных прерываний BIOS.  
Данные функции использовались в ОС подобной MS-DOS и почти не используются в современных ОС. 
 
В ПК первых поколений код записывался в микросхему постоянной памяти ROM - программа записывается в процессе производства данной микросхемы. 
Далее с развитием появляется PROM - изначально создавалась чистой, запись возможна только один раз.  
Далее появляются микросхемы с возможностью перезаписи EPROM - стираемая программируемая память, информация удалялась с помощью специального ультрафиолетового излучения.  
EEPROM - электрически стираемая программируемая память. 
Flash EEPROM - может быть перезаписана с помощью специальных программ прямо на компьютере. 
 
Запись новой версии BIOS называется перепрошивкой. Данная операция может потребоваться, чтобы добавить в код BIOS новые функции, исправить ошибки или заменить поврежденный код. 
 
## Последовательность закрузки компьютера 
 
### **процессы после включчения питания** 
первое устройство после нажатия кнопки питания - блок питания. Если все питающие напряжения окажутся в норме, на системную плану будет подан спец сигнал, свидетельствующий об успешном тестировании блока питания и разрешающий запуск компонентов системной платы. После этого чипсет формирует сигнал сброса центрального процессора, по которому очищаются регисторы процессора, и он запускается. Далее последовательно выполняя команды BIOS процессор начнет процедуру самотестирования и post. 
 
### **процедура post** 
Состоит из следующих этапов: 
1. Первоначальная инициализация основных системных компонентов. 
2. Детектирования оперативной памяти, копирование кода BIOS в оперативную память и проверка контрольных сумм BIOS. 
3. Первоначальная настройка чипсета.  
4. Поиск и инициализация видео адаптера. Современные видео адаптеры имеют собственную BIOS, которую системный BIOS пытается обнаружить. В ходе инициализации видео адаптера на экране появляется первое изображение, сформированное с помощью BIOS видео адаптера.  
5. Проверка контрольных сумм Cmos и состояния батарейки. Если контрольная сумма Cmos ошибочна, будут загружены значения по умолчанию. 
6. Тестирование процессора и оперативной памяти.  
7. Подключение клавиатуры, тестирование портов ввода-вывода и других устройств.  
8. Инициализация дисковых накопителей.  
9. Распределение ресурсов между устройствами. 
10. Поиск устройств, имеющих собственный BIOS. 
11. Вызов программного прерывания BIOS, который ищет загрузочный сетктор на устройствах, указанных в списках загрузки. 
Далее происходит загрузка ОС.8
